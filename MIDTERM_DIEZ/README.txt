WEB BASE CALCULATOR

A simple, user-friendly, and responsive web-based calculator that supports basic arithmetic operations, percentage calculation, square root function, backspace, and keyboard input.

HOW TO USE:

Enter Numbers: Click on the number buttons (0-9) or use your keyboard.
Click on operation buttons (+, -, *, /) or type them.
Use Special Functions: %: Converts the number into a percentage (50% → 0.5). √: Calculates the square root of the current number. ← (Backspace): Removes the last digit entered.
Click = or press Enter.
To clear display, press C or Escape to reset.
